# Sales Data Analysis Project
The main objective of this project is to understand sales performance, profit distribution, and regional trends using data analysis techniques.  The analysis helps identify top-performing products, profitable regions, and overall business performance, which can support data-driven decision-making.

# Dataset Overview

The dataset includes the following key attributes:

Order_ID – Unique identifier for each order

Order_Date – Date of the order

Region – Sales region (North, South, East, West)

Product – Product name

Category – Product category (Electronics, Furniture, etc.)

Sales_Amount – Total sales value

Profit – Profit earned per order

Quantity – Number of units sold
